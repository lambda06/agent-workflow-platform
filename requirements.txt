# PostgreSQL — psycopg3 (psycopg[binary]) replaces asyncpg across the whole stack.
# It is the same driver used internally by LangGraph's AsyncPostgresSaver, so agents
# and the LangGraph checkpointer now share one driver with zero protocol conflicts.
# asyncpg was removed: it uses a different wire-protocol implementation from psycopg3
# and caused SSL negotiation failures when both ran against the same Supabase instance.
# sqlalchemy[asyncio] was removed: it was never used — raw psycopg3 is sufficient for
# the lean JSONB inserts this project requires.
